##Running The Program
In order to run the program, please go to the src folder of the submission. In order to compile, type javac LBSMain.java. In order to run the code, type java LBSMain <argument1><argument2><argument3> where argument 1 is the functionality you would like to use such as CHECK or SOLVE , argument 2 is the problem instance and if necessary argument 2 is the solution which is only needed when using the checking functionalities. More details can be found in the printUsage method of LBSMain.
